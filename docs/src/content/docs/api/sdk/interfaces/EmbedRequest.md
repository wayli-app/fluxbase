Request for vector embedding generation

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| `model?` | `string` | Embedding model to use (defaults to configured model) |
| `text?` | `string` | Text to embed (single) |
| `texts?` | `string`[] | Multiple texts to embed |
